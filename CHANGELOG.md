# Changelog

## [2.0.0] - 2025-12-05

### 🎉 全新重构 - Coding Buddy 编程搭子

完全重新设计的插件，从"挂机游戏"转型为"编程伴侣"。

### ✨ 新功能

- **🐱 搭子系统**
  - 搭子会根据你的编程状态变化表情
  - 心情和能量系统
  - 互动功能：抚摸、喂食、玩耍
  - 7 种皮肤可解锁

- **💬 聊天系统**
  - 预设对话，开箱即用
  - 可选接入 AI（OpenAI/Claude）
  - 快捷指令：/stats, /ddl, /focus
  - 搭子会主动打招呼、提醒休息

- **🍅 专注模式**
  - 番茄钟工作法
  - 可配置工作/休息时长
  - 完成后搭子会庆祝

- **📋 DDL 管理**
  - 添加截止日期任务
  - 临近时搭子会着急提醒
  - 过期任务警告

- **📊 成长记录**
  - 今日代码行数
  - 专注时长统计
  - 连续编程天数

### 🗑️ 移除功能

- 金币系统
- 升级购买
- 战斗系统
- 抽奖系统
- 翻译功能
- 代码截图
- 编码特效（粒子、波纹）

### 🔧 技术改进

- 简化代码架构
- 减少文件数量
- 更清晰的模块划分
