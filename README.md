# Coding Buddy - 编程搭子 🐱

一个住在你 VSCode 里的小伙伴，陪你写代码、管理时间、记录成长。

## ✨ 功能特点

### 🐱 搭子陪伴
- 搭子会根据你的状态变化表情
- 写代码时认真工作，长时间不动会打瞌睡
- 完成任务会庆祝，DDL 临近会着急提醒
- 可以抚摸、喂食、陪玩，增加心情和能量

### 💬 智能聊天
- 预设对话：开箱即用，无需配置
- AI 对话：可选接入 OpenAI/Claude，真正能聊天
- 快捷指令：`/stats` 查看统计，`/ddl` 查看任务，`/focus` 开始专注

### 🍅 专注模式
- 番茄钟工作法，默认 25 分钟工作 + 5 分钟休息
- 完成 4 个番茄钟后自动长休息
- 搭子会安静陪伴，完成后一起庆祝

### 📋 DDL 管理
- 添加截止日期任务
- 临近时搭子会着急提醒
- 完成后搭子会开心庆祝

### 📊 成长记录
- 今日代码行数
- 专注时长统计
- 连续编程天数
- 皮肤收集解锁

## 🚀 快速开始

1. 安装插件后，左侧活动栏会出现 ❤️ 图标
2. 点击打开编程搭子面板
3. 状态栏会显示搭子状态和 DDL 提醒

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd+Alt+B` | 打开编程搭子 |
| `Cmd+Alt+F` | 开始专注 |
| `Cmd+Shift+D` | 添加 DDL |

## 💬 聊天指令

| 指令 | 功能 |
|------|------|
| `/stats` | 查看今日统计 |
| `/ddl` | 查看 DDL 列表 |
| `/focus` | 开始专注模式 |

或者直接用自然语言：
- "今天写了多少代码"
- "还有什么 DDL"
- "开始专注"
- "休息一下"

## 🎨 皮肤解锁

| 皮肤 | 解锁条件 |
|------|----------|
| 😺 默认小猫 | 默认拥有 |
| 😴 瞌睡猫 | 专注 500 分钟 |
| 😎 酷猫 | 连续编程 7 天 |
| 🌟 明星猫 | 累计 5000 行代码 |
| 🥷 忍者猫 | 完成 20 个 DDL |
| 🤖 机器猫 | 连续编程 14 天 |
| 🥳 派对猫 | 专注 1500 分钟 |

## ⚙️ 配置

在设置中搜索 `coding-buddy`：

- `focus.workMinutes`: 专注时长（默认 25 分钟）
- `focus.breakMinutes`: 短休息时长（默认 5 分钟）
- `focus.longBreakMinutes`: 长休息时长（默认 15 分钟）
- `ai.provider`: AI 提供商（none/openai/claude/custom）
- `ai.apiKey`: AI API 密钥

## 🤖 AI 聊天配置（可选）

如果想让搭子更智能，可以配置 AI：

1. 打开设置，搜索 `coding-buddy.ai`
2. 设置 `provider` 为 `openai` 或 `claude`
3. 填入你的 API Key

配置后，搭子就能：
- 回答代码问题
- 帮忙 debug
- 闲聊解压
