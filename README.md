# Coding Idle Game

一个让编程更有趣的 VSCode 扩展！结合挂机游戏、酷炫特效和实用工具，让你在编码时赚取金币、管理任务、生成精美代码截图。

## ✨ 核心功能

### 🎮 挂机游戏
- 编码时自动产生金币
- 购买升级项提升产出速度
- 解锁成就徽章
- 抽奖系统赢取奖励

### 🐱 编程搭子
你的专属编程伙伴，显示在状态栏：

- **娱乐模式**：`😊 Lv.5 ❤️85% ⚡90%` - 显示搭子状态
- **工作模式**：`⏰ DDL: 2小时后` - 只显示DDL（适合公司）
- **专注模式**：`🔔` - 极简显示（深度专注）

**功能**：
- ⏰ DDL任务管理和提醒
- 📸 代码截图生成（可复制/下载）
- 🎨 多种皮肤解锁
- ❤️ 互动系统（抚摸/喂食/玩耍）

### 💥 编码特效
- 🪙 金币粒子：输入代码时爆出金币
- ✨ 关键词爆炸：输入 `function`、`class` 等触发特效
- 🌊 波纹效果：点击时产生彩色波纹
- 完全可自定义关键词和符号

### 🌐 翻译功能
- 选中文本快速翻译（中英互译）
- 智能生成变量名（支持多种命名风格）
- 右键菜单快捷访问

## 🚀 快速开始

### 安装

**方式1：Marketplace**（推荐）
1. 打开 VSCode/Cursor
2. 搜索 "Coding Idle Game"
3. 点击安装

**方式2：扩展ID**
```
xinzhixu.funny-vscode-extension
```

### 基础使用

#### 1. 打开游戏
点击左侧 **⭐ 挂机游戏** 图标

#### 2. 查看搭子
状态栏左侧自动显示搭子信息
- 点击可打开详细面板
- 命令：`🔄 切换搭子显示模式` 切换工作/娱乐模式

#### 3. 生成代码截图
1. 选中代码
2. 右键 → `📸 生成代码截图`
3. 点击 `快速生成`
4. 点击 `📋 复制图片` 或 `💾 下载PNG`

#### 4. 添加DDL
命令面板（`Cmd+Shift+P`）→ `⏰ 添加DDL任务`

#### 5. 翻译文本
1. 选中文本
2. 右键 → `🌐 翻译选中文本` 或 `💡 变量名建议`

## 🎯 常用命令

按 `Cmd+Shift+P` 打开命令面板：

**搭子相关**：
- `🔄 切换搭子显示模式` - 娱乐/工作/专注模式切换
- `📸 生成代码截图` - 生成精美代码截图
- `⏰ 添加DDL任务` - 添加截止时间提醒
- `❤️ 与搭子互动` - 抚摸/喂食/玩耍

**游戏相关**：
- `💰 手动获得金币` - 点击获得金币
- `💾 备份存档` - 备份游戏数据

## 🐛 常见问题

**Q: 状态栏没有显示搭子？**  
A: 重新加载窗口（`Cmd+R`）或重启VSCode

**Q: 代码截图生成失败？**  
A: 尝试切换渲染方式（HTML ↔ Canvas）

**Q: 特效没有显示？**  
A: 在侧边栏中启用"💥 编码特效"开关

**Q: 翻译功能如何配置？**  
A: 侧边栏 → 设置标签 → 配置翻译API → 填入百度翻译密钥（[免费申请](https://fanyi-api.baidu.com/doc/21)）
